# us-retail-insights-dashboard
This project involves developing a 'Customer Insights Dashboard' using Python and Matplotlib to analyze and visualize key customer-related metrics that support data-driven decision-making within the retail organization.
Generated: 
Line plot displaying 'Monthly Revenue Trend'
Bar Plot displaying 'City wise sales'
Pie Chart displaying 'Product category wise sales distribution'
Donut Chart displaying 'Payment method distribution'
Histogram to display the 'Customer order frequency'
